# Library Management System - Nanded

A comprehensive municipal library management system for Nanded city libraries.

## 🚀 Demo Project

This is a demo/prototype application showcasing a modern library management system with features for administrators, librarians, and citizens.

## ✨ Features

- **Multi-Role Access**: Admin, Librarian, and Citizen roles
- **Book Management**: Search, browse, and manage physical & digital books
- **Circulation**: Issue and return books with automatic fine calculation
- **Member Management**: Register and manage library members
- **Facility Check-In**: Track member visits to library facilities
- **Events Management**: Create and manage library events
- **Digital Resources**: Download PDFs and digital content
- **Reports & Analytics**: Comprehensive dashboards and metrics
- **Accessibility**: Built-in accessibility features for all users
- **Multi-Library Support**: Manage multiple library branches

## 🏗️ Tech Stack

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **UI Framework**: Tailwind CSS
- **UI Components**: Radix UI + shadcn/ui
- **Routing**: React Router v6
- **State Management**: React Context + React Query
- **Icons**: Lucide React
- **Charts**: Recharts

## 📱 Key Features by Role

### Admin
- Manage all libraries and branches
- View city-wide analytics
- Approve member registrations
- Manage system settings
- Generate reports

### Librarian
- Issue and return books
- Check-in/check-out members (facility entry)
- Manage library members
- Handle fines and fees
- Create library events
- View library-specific reports

### Citizen
- Search and browse books
- Request book borrowing
- View borrowed books
- Pay fines online
- Register for events
- Download digital resources
- Track membership status
